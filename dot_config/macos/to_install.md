# Apps To Install

Common macOS apps and tools.

## Install Homebrew

- Website: https://brew.sh/
- Install:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

- Install with Homebrew:
  - `alfred` (cask)
  - `aerospace` (cask)
  - `font-hack-nerd-font` or any nerd font (cask)
  - `iterm2` (cask)
  - `bat`
  - `chezmoi`
  - `eza`
  - `fd`
  - `fzf`
  - `git`
  - `git-delta`
  - `glow`
  - `jq`
  - `lazygit`
  - `neovim`
  - `sesh`
  - `television`
  - `tldr`
  - `tmux`
  - `zoxide`
